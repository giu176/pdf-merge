Installazione:
1) (internet necessario) installa l'ultima versione di Ubuntu da Microsoft Webstore, segui l'installazione e scegli un NOMEUTENTE e una password.
2) In Windows apri Esplora Risorse e incolla nella cartella \\wsl.localhost\Ubuntu\home\NOMEUTENTE i file "pdf.py", "install.sh" e "initialize.sh"
3) Avvia Ubuntu e accedi con le credenziali scelte in fase di installazione.
4) (internet necessario) esegui:
	sudo sh /home/NOMEUTENTE/install.sh
5) (internet necessario) esegui:
	sh /home/NOMEUTENTE/initialize.sh

Procedura per creare i pdf con copertina:
1) Avvia Ubuntu e, se richiesto, accedi con le credenziali scelte in fase di installazione.
2) Esegui una volta:
	source /home/NOMEUTENTE/scripts/bin/activate && cd /home/NOMEUTENTE
3) In Windows apri Esplora Risorse e incolla nella cartella \\wsl.localhost\Ubuntu\home\NOMEUTENTE i file "template.docx" e il pdf con copertina ACCA (input.PDF) da modificare.
4) Modifica il file template.docx come desiderato (lasciare l'ultima pagina vuota con solo intestazione e piè di pagina).
5) Creare il file template.pdf con Word.
6) Esegui:
	python3 pdf.py template.pdf input.PDF output.pdf
	
Dove:
	- template.pdf è la copertina del documento con di seguito eventuali pagine da inserire all'inizio del documento e una pagina vuota alla fine (con solo intestazione e piè di pagina)
	- input.pdf è il file generato con primus/mantus ecc.. sostituire con il nome del del file da modificare ad esempio "computo.PDF" (è possibile digitare le iniziali del nome file e premere il tasto TAB per autocompletamento). La prima pagina di questo pdf verrà rimossa e sostituira con quelle del template.pdf
	- output.pdf è il nome del file che verrà generato, sostituire con il nome desiderato tra apici ad esempio "25-03-10 I&C-PRS01-PE-24-066 Computo Metrico Estimativo.pdf"

Il programma elimina la prima pagina dal file input.PDF e inserisce in coda al file template.pdf tutte le rimanenti pagine ridimensionate e complete di intesazione e piè di pagina.
Il file output.pdf viene salvato nella cartella \\wsl.localhost\Ubuntu\home\NOMEUTENTE. Se non compare aggiornare la cartella (tasto DX>Mostra altre opzioni>Aggiorna).
